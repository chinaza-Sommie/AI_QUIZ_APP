# AI Quiz App
This is an AI powered quiz app that allows individuals to take quizzes on their selected stack thereby allowing them to identify knowledge gaps based on the analysis done on the users' quiz results.
<br/><br/>
## Aim

This App is Aimed at:
- Allowing users to take a quiz on their selected stack
- Allowing users to find out the knowledge gaps and find out what they need to work on
- Provide analysis to users with knowledge gaps and how to fill them
<br/> <br/>
## Features
The Features include:
- A quiz feature that runs for each stack selected
- AI Engine that generates the questions and corresponding answers
- AI provided analysis on the results
- Grading feature
- Emailing feature that sends responses to the users email
